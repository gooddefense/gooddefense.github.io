创建Github仓库

  • 登录GitHub账号，如果没有的话就先注册一个
  • 登录成功之后，进入到个人主页，点击Repositories,这一个页面就是仓库；点击右上角的New按钮可以进入创建仓库界面，自行填写相关参数就可以了
