Git的工作流程

  • 克隆 Git 资源作为工作目录。
  • 在克隆的资源上添加或修改文件。
  • 如果其他人修改了，你可以更新资源。
  • 在提交前查看修改。
  • 提交修改。
  • 在修改完成后，如果发现错误，可以撤回提交并再次修改并提交。

Git基本操作

Git的作用是什么？Git 的工作就是创建和保存你项目的快照及与之后的快照进行对比。

常用的命令有以下六个命令：

  • git clone
  • git push
  • git add
  • git commit
  • git checkout
  • git pull
